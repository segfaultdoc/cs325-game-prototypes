Random Fun

Use up and down keys, the spacebar is the brakes

  • Originality: What is original about my game is the irony of an airplane collecting jelly fish sandwiches in the ocean.
  • Technical Merit: For this game I decided to use some more advance javascript object oriented programming. I have modularized my source files by using an Enemy object and blob object. Originally I attempted to use ES6 syntax in order to facilitate inheritance, but this was proving to be far too complex with github pages.
  • Prototype Postmortem: What did you learn from this prototype? The physics can get very confusing. When things seem to be working they can break in the blink of an eye. For example I'm supposed to have 12 jelly fish on the screen and I've done everything right, but it's not working. Mainly I've got a better understanding of the phaser libraries.
